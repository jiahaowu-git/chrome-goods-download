**requestPageData() function:**
```javascript
function requestPageData() {
  chrome.runtime.sendMessage({ type: 'getPageData' }, (response) => {
    if (chrome.runtime.lastError) {
      console.error('Failed to request page data:', chrome.runtime.lastError.message);
    }
  });
}
```

**onMessage listener:**
```javascript
chrome.runtime.onMessage.addListener((message, sender, sendResponse) => {
  if (message.type === 'pageData') {
    handlePageData(message.data);
    return true;
  }
  return true;
});
```

Add click handler for download button and selection counting:

**updateSelectedCount() function:**
```javascript
function updateSelectedCount() {
  const total = mainImages.filter(i => i.checked).length + detailImages.filter(i => i.checked).length;
  const countEl = document.querySelector('.image-count');
  if (countEl) {
    countEl.textContent = total > 0 ? `已选 ${total} 张` : '';
  }
  // Update download button state
  const btn = document.querySelector('.download-btn');
  if (btn) {
    btn.disabled = total === 0;
  }
}
```

**collectSelectedImages() function:**
```javascript
function collectSelectedImages() {
  const selected = [
    ...mainImages.filter(i => i.checked),
    ...detailImages.filter(i => i.checked)
  ];
  return selected;
}
```

**Download button click handler:**
```javascript
document.querySelector('.download-btn').addEventListener('click', () => {
  const selected = collectSelectedImages();
  if (selected.length === 0) return;
  // Dispatch download event (Phase 4 implementation)
  window.dispatchEvent(new CustomEvent('downloadImages', {
    detail: { images: selected, pageTitle: document.title }
  }));
});
```

**Update checkbox toggle logic to call updateSelectedCount():**
- After toggling any checkbox, call updateSelectedCount()
- This updates both the count display AND button disabled state

**Button disabled state CSS:**
- .download-btn[disabled] has opacity 0.5 and cursor not-allowed
- Button starts disabled (opacity 0.5) until images load and at least 1 is checked

**Download event (Phase 4 integration point):**
- Window CustomEvent 'downloadImages' with detail.images array
- Phase 4 listener will handle the actual download (zip creation, etc.)
- This plan only wires the dispatch — Phase 4 implements the handler
