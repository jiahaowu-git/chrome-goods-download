# Plan: 05-01 Integration Testing

## Objective

Validate the complete extension workflow on real Taobao/Tmall product pages.

## Tasks

### Task 1: Test main image extraction on real Taobao page

**Read first:**
- content.js (image extraction logic)
- sidepanel.js (rendering and message handling)

**Action:**
- Open a real Taobao product detail page (e.g., `https://item.taobao.com/item.htm?id=xxxx`)
- Click extension icon to open side panel
- Verify main images are detected and displayed (expect: 5 images)
- Check that image count badge shows "已选 X / X 张"

**Acceptance criteria:**
- Main images grid renders without empty placeholder
- All main images display as square thumbnails
- Checkboxes are checked by default

### Task 2: Test detail image extraction on real Taobao page

**Read first:**
- content.js (extractDetailImages function)

**Action:**
- On the same Taobao page, verify detail/description images load
- Check count badge matches actual images in DOM
- Verify section expands/collapses correctly

**Acceptance criteria:**
- Detail images count badge shows correct number
- Section scrolls properly when expanded (no clipping)
- All detail images render in 2-column grid

### Task 3: Test download workflow

**Read first:**
- sidepanel.js (downloadImages event listener, fetchImageAsJpgBlob, zip generation)

**Action:**
- Select a subset of images using checkboxes
- Click "一键下载" button
- Verify zip file downloads with correct product title as filename
- Extract zip and verify:
  - Two folders: "主图" and "详情图"
  - Images are numbered 001.jpg, 002.jpg, etc.
  - All selected images are present

**Acceptance criteria:**
- Zip file downloads successfully
- Zip contains two folders (主图, 详情图)
- Filename uses product title (not page title)
- JPG conversion works (files are valid JPG)

### Task 4: Test on real Tmall page

**Read first:**
- content.js (detectPlatform, isDetailPage)

**Action:**
- Open a real Tmall product detail page (e.g., `https://detail.tmall.com/item.htm?id=xxxx`)
- Repeat Tasks 1-3 on Tmall

**Acceptance criteria:**
- Platform detection works for tmall.com
- Main images extract correctly
- Detail images extract correctly
- Download works with Tmall product title

### Task 5: Test error handling on non-ecommerce page

**Read first:**
- content.js (sendUnsupportedMessage)
- sidepanel.js (handlePageData error branch)

**Action:**
- Open a non-product page (e.g., taobao.com homepage, or a blog)
- Click extension icon
- Verify error message displays: "无法获取图片：该页面不是商品详情页"

**Acceptance criteria:**
- Error state shows clear message
- Empty state renders in both main and detail grids
- No console errors in service worker or content script

### Task 6: Verify extension icons display correctly

**Read first:**
- manifest.json (icons configuration)

**Action:**
- Check chrome://extensions/ page
- Verify extension icon shows at all sizes (16, 32, 48, 128)
- Verify toolbar icon matches icons/logo.png

**Acceptance criteria:**
- All icon sizes render without broken image placeholder
- Icon is visible in Chrome toolbar

## Verification

- Open the extension on a real product page and confirm all UI elements work
- Check for any console errors in service worker or content script
- Verify no CSP violations in browser console

## Must-haves

- Extension loads without errors in chrome://extensions/
- Side panel opens via toolbar icon click
- Main images display in 2-column grid with checkboxes
- Detail images display in 2-column grid with checkboxes
- "一键下载" generates zip with 2 folders (主图, 详情图)
- Zip filename uses product title from page
- Error message shows on non-product pages
